# Talos K8s Home Lab

Reproducible Talos Kubernetes cluster for home lab environments.

## Overview

This documentation covers the setup, configuration, and operation of a Kubernetes cluster using [Talos Linux](https://www.talos.dev/).

## Features

- Reproducible infrastructure as code
- Declarative cluster configuration
- Secure by default
